Для запуска программы поиска нужно запустить Seeker.exe

ВАЖНО!!! для отображения русского шрифта в программе в свойствах консоли(правой кнопкой мыши по шапке окна) установить шрифт Lucida Console

Программа работает следующим образом: сначала создается словарь с обратным индексом. При появлении сообщения о завершении создании словаря можно вводить поисковый запрос. Запрос должен состоять только из слов разделенных пробелами, без каких-либо символов пунктуации. После ввода строки программа отображает до 10 вариантов поиска после чего можно снова вводить строку для поиска. Для выхода из программы нужно ввести в поисковый запрос exit

Что касается остальных файлов:

input.txt - тут находятся настройки программы которые задают количество файлов и их название, в теории можно добавлять в папку свои текстовые файлы, редактировать файл input.txt (добавив названия файлов и изменив, если надо, значение их количества) и в итоги осуществлять поиск по ним.

output.txt - словарь с обратным индексом созданный в начале работы программы. Составлен по такому шаблону:
<стеммированое слово>
<имя файла> <кол-во слов в файле> <tf-idf>
<список координат этого слова в файле>

Linker.cpp - Листинг осеовной программы

stem.cpp - Ресурсный файл для стэма
